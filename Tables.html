Tables

<table>: The <table> element is used to create an HTML table. Tables are used to organize data into rows and columns. It acts as a container for all other table elements. <tr> (Table Row): The <tr> element represents a table row. Rows are used to group a set of table data or table header cells together. Each <tr> element typically contains one or more <td> (table data) or <th> (table header) elements. <th> (Table Header): The <th> element is used to define a header cell within a table. It represents a header for a column or a group of columns in a table. Header cells are typically displayed in bold and centered by default. <td> (Table Data): The <td> element is used to define a standard data cell within a table. It represents a single piece of data within a row and column of the table. Data cells are typically displayed with regular text formatting. <thead> (Table Head): The <thead> element is used to define the header section of a table. It contains a group of table rows (<tr>) that are considered the table's header. The content within the <thead> typically consists of <th> (table header) elements, which represent the headers for various columns in the table. These header cells are often displayed in bold and centered by default. </thead> <tbody> (Table Body): The <tbody> element groups a set of table rows into a table body. While it's not required in HTML, it is often used to logically group rows within a table, especially when dealing with large tables or when styling the table with CSS. <tfoot> (Table Foot): The <tfoot> element represents the footer section of a table. It's typically used to define summary or footer information for a table, such as totals, subtotals, or additional notes. The <tfoot> element is placed after the <tbody> element and before the closing </table> tag.

Example

            
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1, Row 1</td>
            <td>Data 2, Row 1</td>
        </tr>
        <tr>
            <td>Data 1, Row 2</td>
            <td>Data 2, Row 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Total:</td>
            <td>Sum of Data 2</td>
        </tr>
    </tfoot>
</table>

</body>
</html>

With Borders:

To add a border, use the CSS border property on table, th, and td elements:

Example

            
<!DOCTYPE html>
<html>
<head>
    <style>
        table, th, td {
            border: 1px solid black; /* Add a border to the table, th, and td elements */
        }
    </style>
</head>
<body>

<h2>Table with Borders</h2>

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
        </tr>
    </tfoot>
</table>

</body>
</html>

Collapsed Table Borders

To avoid having double borders like in the example above, set the CSS border-collapse property to collapse. This will make the borders collapse into a single border:

Example

            
<!DOCTYPE html>
<html>
<head>
<style>
table, th, td {
    border: 1px solid black;
    border-collapse: collapse;
}
</style>
</head>
<body>

<h2>Collapsed Borders</h2>
<p>If you want the borders to collapse into one border, add the CSS border-collapse property.</p>

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>

    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
        </tr>
    </tfoot>
</table>

</body>
</html>

Style Table Borders

If you set a background color of each cell, and give the border a white color (the same as the document background), you get the impression of an invisible border:

Example

            
<!DOCTYPE html>
<html>
<head>
<style>
table, th, td {
    border: 1px solid white;
    border-collapse: collapse;
}
th, td {
    background-color: #96D4D4;
}
</style>
</head>
<body>

<h2>Table With Invisible Borders</h2>

<p>Style the table with white borders and a background color of the cells to make the impression of invisible borders.</p>

<table style="width:100%">
    <tr>
        <th>Firstname</th>
        <th>Lastname</th> 
        <th>Age</th>
    </tr>
    <tr>

        <td>John</td>
